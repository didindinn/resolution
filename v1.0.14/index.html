@unstoppabledomains/resolution

Resolution

A library for interacting with blockchain domain names.

You can use Resolution in a <script> tag from a CDN, or as the Resolution package on the npm registry.

Supported domain zones:

  • .zil
  • .eth
  • .test
  • .xyz
  • .luxe
  • .crypto
